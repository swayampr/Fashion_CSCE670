# Fashion?!
Project for CSCE 670 course.

## How to run

1. Download the [Polyvore](https://www.kaggle.com/datasets/dnepozitek/maryland-polyvore-images/data) dataset. Put the unzipped files into `images` directory in the `data` folder.
2. Train

   ```sh
   cd main
   python train.py
   ```

3. Evaluate

   ```
   python evaluate.py
   ```
